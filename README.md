## TG 频道能看到什么内容

订阅了没有什么信息密度的频道是在浪费通知权限。DMIT 的 TG 频道实际上发什么？

翻了一下近期的记录，主要集中在这几类：

1. **网络状态公告** — 哪个节点有异常、预计恢复时间、已解决通知
2. **硬件维护通知** — 比如把香港机器的 CPU 从 7402 平台换成 7003 平台，都会提前告知
3. **DDoS 攻击应对说明** — 2025 年底 HKG 和 TYO 机房受到持续攻击，TG 频道全程有进度更新和补偿公告
4. **新产品上线** — 洛杉矶 Eyeball 系列、圣何塞 T1 系列都是先在频道公告的
5. **线路调整通知** — 比如 AS9929 异常、GSL 接入状态变化这类偏技术的更新

说实话，不是每条你都看得懂，部分内容偏向运维层面。但如果你机器上有在跑服务，那维护通知这块确实有用，省得遇到卡顿了还傻乎乎开工单问。

## FAQ

**Q：DMIT TG 频道是不是官方的，会不会是假冒的？**

A：@DMIT_INC 是 DMIT 官方维护的英文主频道，@DMIT_INC_CN 是他们的中文 NOC 日志频道。两个都是只读广播，没有互动功能，发的内容和官网维护公告是对应的。

**Q：订阅 TG 频道能拿到独家优惠码吗？**

A：历史上有过，但不是常规操作。促销码出现时机和产品发布或特殊活动绑定，不是持续发放的。在频道里看到的码，官网验证通过再用，过期了就是过期了。

**Q：DMIT 有没有 TG 客服或者可以在频道里提问？**

A：官方频道是单向广播，不能回复。需要技术支持走官网工单系统，有中文客服。

**Q：买 DMIT 之前能不能先测一下网络质量？**

A：可以，DMIT 官方提供各机房的测试 IP。LAX.EB 系列的测试 IP 是 154.17.226.2（也有 IPv6 版本），在买之前 ping 一下或者跑个 traceroute 心里大概有数。

**Q：优惠码是不是只能用一次？**

A：DMIT 的大部分循环优惠码是每个账户首次购买时使用，折扣在续费时持续生效，不是一次性码，但一个账户通常只能使用一次注册。续费用同一个码折扣自动延续。
